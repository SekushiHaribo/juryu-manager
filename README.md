# 주류 관리 (juryu-manager)

요식업장 냉장고의 주류·음료 채울 개수를 관리하는 모바일 웹앱(PWA)입니다.

- 냉장고 칸 배치(층·칸·여러 bay), 주류 카테고리/용량/색상 설정
- 일반 모드에서 빠진 개수를 숫자 또는 드래그로 입력
- 기록 카드와 합계 탭에 당일 입력값 자동 집계
- 데이터는 사용 기기의 브라우저(localStorage)에 저장됩니다

## 사용 방법
배포된 링크를 휴대폰 브라우저로 열고 **공유 → 홈 화면에 추가** 하면 앱처럼 사용할 수 있습니다.

## 배포 (GitHub Pages)
저장소 **Settings → Pages → Branch: `main` / root** 로 설정하면
`https://<사용자명>.github.io/juryu-manager/` 에서 열립니다.
